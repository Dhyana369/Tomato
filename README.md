# Tomato - Food Ordering & SQL Analytics App

Live Demo: http://Dhyana5.pythonanywhere.com/
---

## Overview
A full-stack food ordering application built primarily to showcase SQL skills, relational database design, and analytical queries.
---

## Features
- Add to cart and place orders
- MySQL-backed backend with normalized schema
- SQL analytics (revenue, customers, categories)
- Flask-based API and UI
---

## Tech Stack
Python, Flask, MySQL, HTML, CSS, JavaScript
---

## SQL Highlights
- Complex JOIN queries
- Aggregations (SUM, COUNT)
- Revenue & customer analysis
---
## Project Structure
- `app_index.py` – Flask backend
- `templates/` – UI
- `static/` – images & styling
- `data_sql.sql` – schema + queries
- `data_sql.ipynb` – SQL analysis
